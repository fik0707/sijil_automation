1. Create a Google Form for Certification

Form Setup

Configure Your Google Form

  1. Open Google Forms and create a Blank Form.
  2. Add the form title, for example: Sijil Kursus......
  3. Create 1 question:
    NAMA PENUH (HURUF BESAR) (Short answer), and make it mandatory (Required: press on the slide bar).
  4. Click Settings at the top of the form.
  5. Expand the "Responses" section. Click on the options next to "Collect email addresses" and choose "Responder input".
  6. Click Send or Publish (top right corner).

6. Set Up Automation Trigger

Triggers

Create Submit Trigger

This tells Google to automatically execute our script whenever a participant submits a new Google Form response.

  1. Move your cursor to the left and click on the clock icon (red circle) Trigger.
  2. Click the blue + Add Trigger (+ Tambah Pencetus) button in the bottom right.
  3. Configure the trigger settings exactly as follows:
    • Choose which function to run: sendCertificate
    • Choose which deployment should run: Head
    • Select event source: From spreadsheet (Dari helaian)
    • Select event type: On form submit (Semasa menghantar borang)
  4. Click Save.
  5. A window will pop out. Scroll it to the bottom and click Allow.
  • ✓
    Alarm Clock Icon Clicked Opened the Triggers dashboard in the script editor.
  • ✓

7. Run Script

Run Script

Initial Authorization Run

Google requires you to manually approve the permissions for the script to access your Sheets, Drive, Slides, and send emails.

  1. double click on the (< >) symbol on the left side.
  2. Select sendCertificate in the dropdown menu next to the "Debug" button, then click Run (Jalan).
  3. Wait for the execution log to be completed. It will appear at the bottom:
    Notice Execution completed
  • ✓
    Script Run Executed Allowed access permissions and ran the script.

⚠️ Important Note

The execution log will show an error Ralat: Tiada data... if your Sheet is completely empty during the test run. This is normal! As long as the execution completed, you are ready.
